# Social assets

Generated social-network assets for PARANOIA.

- `linkedin-profile.png` — 400x400 profile/logo image.
- `linkedin-cover.png` — 4200x700 LinkedIn Page cover.
- `linkedin-share.png` — 1200x627 LinkedIn shared-link image, promoted from `imagegen-tests/source-rich.png`.
- `x-profile.png` — 400x400 X profile image.
- `x-header.png` — 1500x500 X profile header.
- `x-card.png` — 1200x600 X summary_large_image card, promoted from `imagegen-tests/source-rich.png`.

SVG files are retained as editable fallbacks for cover/header/card/share variants. Feed assets use the imagegen raster direction because it is more distinctive.
