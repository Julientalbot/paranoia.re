# Social assets

Generated social-network assets for PARANOIA.

- Root PNG exports: `../logo.png`, `../logo-mark.png`, `../favicon.png`.
- `linkedin-profile.png` — 400x400 profile logo, generated from `logo-mark.svg`.
- `linkedin-cover.png` — 4200x700 LinkedIn Page cover.
- `linkedin-share.png` — 1200x627 LinkedIn shared-link image.
- `x-profile.png` — 400x400 X profile logo, generated from `logo-mark.svg`.
- `x-header.png` — 1500x500 English X profile header, retouched for the profile-photo safe area.
- `x-card.png` — 1200x600 X summary_large_image card.

SVG files are retained as editable fallbacks for cover/header/card/share variants. LinkedIn shared-link SVG/PNG exports are generated by `tools/linkedin-visual-refresh/generate.mjs`.
